## Story Post {#story}

### Personal Failure → Lesson

```
I got rejected from [number] [jobs/companies/opportunities].

It crushed me.

I thought I wasn't good enough.

But here's what nobody told me:

[Rejection/failure] isn't about you.

It's about [insight - fit, timing, circumstances].

The [number]th try?

[Positive outcome].

What changed wasn't me.

It was my [approach/mindset/strategy].

If you're facing rejection right now:

→ [Advice point 1]
→ [Advice point 2]
→ [Advice point 3]

[Encouraging closing line]

What rejection taught YOU the most?
```

## Technical/Tutorial Post {#technical}

### Quick Tip Format

```
Stop doing [bad practice].

Do this instead:

[Better approach in 1 line]

Why?

[Reason 1 - 1 line]
[Reason 2 - 1 line]

Example:

[Bad code/approach]
↓
[Good code/approach]

Result: [Benefit]

Save this for later.

What [topic] tips would you add?
```
